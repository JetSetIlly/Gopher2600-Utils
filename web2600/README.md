# Web2600

Web2600 is a proof-of-concept version of `Gopher2600`, suitable for running in a
web browser. Do note that performance is currently very poor.

To run `Web2600` using a lightweight, development webserver:

	> make webserve

The server will be listening on port 2600.

Note that you need a file in the `web2600/www` folder named "example.bin" for
anything to work.
